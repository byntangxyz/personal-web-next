# Personal Website V3 


dibuat untuk menggantikan personal website v2.1 dari vue.js dan vite. Sekarang saya menggunakan NextJs untuk frameworknya karena saya baru tau NextJs

Mau copy atau modif sendiri? silahkan !

Step 1: Clone repo

``` 
git clone <link repo>

```
Step 2: Instal dependencies
```
npm install
```
Step 3: Run di localhost
```
npm run dev
```
Step 4: Kalau udah modif langsung build ajah
```
npm run build
```
Step 4.5: Cek hasil build
```
npm run start 
```
Step 5: Publish ke github biar bisa di host vercel
```
git init
git add .
git commit -m "Commit pertama"
git branch -M main
git remote <link repo yang kamu buat>
git push -u origin main
```
Step Final: Login ke vercel lalu buat project > hubungkan project ke repo > website sudah jadi
